# Practice LINQ Task

## Опис
Практична робота з дисципліни  
**«Мова інтегрованих запитів (LINQ)»**

Проєкт присвячений вивченню принципів використання технології LINQ у мові програмування C# для обробки та аналізу колекцій даних.

---

## Мета роботи
Засвоїти основні можливості технології LINQ (Language Integrated Query) та навчитися використовувати її для виконання запитів до колекцій об’єктів у мові C#.

---

## Завдання
- Клонувати template-репозиторій з GitHub  
- Ознайомитися зі структурою проєкту  
- Реалізувати LINQ-запити Query1–Query15  
- Перевірити коректність роботи програми  
- Реалізувати ті самі запити у середовищі LINQPad  
- Проаналізувати результати виконання запитів  

---

## Структура проєкту
- `Program.cs` — основна логіка програми та реалізація LINQ-запитів  
- `FootballGame.cs` — клас, що описує футбольний матч  
- `data/results_2010.json` — файл з даними матчів  

---

## Використані технології
- C#
- .NET 8.0
- LINQ
- LINQPad
- Newtonsoft.Json

---

## Результати
У результаті виконання роботи було реалізовано 15 LINQ-запитів для аналізу футбольних матчів збірних команд, а також отримано навички роботи з LINQPad та JSON-даними.
